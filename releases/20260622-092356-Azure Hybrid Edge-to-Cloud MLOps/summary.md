Azure 최신 업데이트

- **Azure Migrate**가 **GitHub Copilot Modernization**과 통합되어 코드 인사이트 제공 (Azure Updates)
- **Azure Databricks**가 **Microsoft OneLake**에 대한 네이티브 읽기 접근 지원 시작 (Azure Updates)
- **Azure Databricks**가 **Microsoft OneLake**에 데이터 저장 기능 제공, 통합 스토리지 레이어 구현 (Azure Updates)
- **Azure Standard V2 NAT Gateway**가 ICMP Echo 요청 및 응답 지원, 네트워크 문제 해결 용이 (Azure Updates)
- **Azure Functions**의 VS Code 확장에 새로운 프로젝트 템플릿 및 갤러리 제공 (Azure Updates)
- **Log Analytics**의 요약 규칙 경험이 새롭게 제공되어 쿼리 성능 향상 (Azure Updates)
