## 🆕 주요 업데이트
- **Foundry Agents in VS Code**: 코드 중심 관찰성으로 평가·개선 루프를 에디터에서 실행
- **Azure Cosmos DB Agent Kit**: AI 코딩 에이전트에 데이터 모델·쿼리 모범 사례를 내장
- **Azure API Management Unified Model API**: 여러 모델 API 형식을 하나로 묶어 교체·거버넌스를 단순화
- **Foundry Models Model Router Policy**: 모델 라우팅 기준을 Azure Policy로 중앙 통제
- **Voice Live + Foundry Agent Service**: 음성 입출력을 별도 오디오 파이프라인 없이 바로 연결
