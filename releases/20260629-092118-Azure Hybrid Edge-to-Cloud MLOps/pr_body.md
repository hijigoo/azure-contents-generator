# 변경 개요
최신 Azure 업데이트 슬라이드를 추가했습니다. 주요 기능과 서비스의 변화가 포함되어 있어, Azure 사용자들이 최신 정보를 쉽게 파악할 수 있도록 하였습니다.

## 주요 업데이트 🆕
- **Azure Blueprints**의 은퇴 일정이 2027년 1월 31일로 연장됨 (Azure Updates)
- **Application Gateway for Containers**의 AI 게이트웨이 기능이 추가됨 (Azure Updates)
- **Azure NetApp Files** 마이그레이션 도우미가 효율적인 데이터 이전 지원 (Azure Updates)
- **Azure VMware Solution**의 AV36 노드 은퇴 일정이 2027년 9월 30일로 변경됨 (Azure Updates)
- **Azure Migrate**가 GitHub Copilot과 통합되어 코드 평가 제공 (Azure Updates)
