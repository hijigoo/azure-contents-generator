TL;DR
이번 주 Microsoft/Azure 공식 업데이트를 반영해 `Microsoft Foundry를 활용한 Agentic AI` 자료의 마지막 슬라이드를 `Latest Azure Updates`로 교체했습니다. 신규 릴리즈 폴더에 PPT/PDF/PNG 미리보기와 업데이트 원본 데이터를 함께 생성해 바로 리뷰할 수 있도록 정리했습니다.

🆕 주요 업데이트
- **Azure Monitor Simple Log Alerts**: GA 전환으로 기본 로그 알림 구성이 단순화되었습니다.
- **Foundry Agents in VS Code Observability**: Public Preview로 코드 중심 관측 기능이 추가되었습니다.
- **Azure Cosmos DB Agent Kit**: GA 제공으로 에이전트 구현 가속이 가능해졌습니다.
- **Azure SQL (Early-June updates)**: Public Preview 업데이트가 공개되었습니다.
- **Azure Files on macOS with Entra ID**: Public Preview로 보안 접근 시나리오가 강화되었습니다.

✅ 리뷰 체크리스트
- [ ] `Latest Azure Updates` 슬라이드가 마지막 장에 1장만 존재하는지 확인
