- **Azure Monitor Simple Log Alerts** 가 GA로 전환되어 간단한 로그 알림 구성이 쉬워졌습니다. (Azure Updates)
- **Foundry Agents in VS Code Observability** 가 Public Preview로 제공되어 코드 중심 추적이 강화되었습니다. (Azure Updates)
