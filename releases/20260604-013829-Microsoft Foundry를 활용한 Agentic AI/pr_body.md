# 변경 개요 ✨
이번 업데이트에서는 Azure의 최신 기능과 서비스가 일반 제공 및 미리보기로 출시되었습니다. 주요 기능으로는 Azure Monitor의 간편 로그 알림, VS Code의 코드 우선 관측 가능성, Azure Cosmos DB의 에이전트 키트 등이 포함됩니다.

## 주요 업데이트 🚀
- Azure Monitor에서 **간편 로그 알림** 기능이 일반 제공 시작됨. [자세히 보기](https://azure.microsoft.com/updates?id=561978) (Azure Updates)
- VS Code에서 **코드 우선 관측 가능성** 기능이 공개 미리보기로 제공됨. [자세히 보기](https://azure.microsoft.com/updates?id=563197) (Azure Updates)
- Azure Cosmos DB를 위한 **에이전트 키트**가 일반 제공됨. [자세히 보기](https://azure.microsoft.com/updates?id=563022) (Azure Updates)
- Azure SQL에서 **AES 256 대칭 키**를 사용한 투명 데이터 암호화 지원. [자세히 보기](https://azure.microsoft.com/updates?id=563142) (Azure Updates)
- macOS에서 **Entra ID 인증**을 통한 Azure Files 접근 지원 시작. [자세히 보기](https://azure.microsoft.com/updates?id=565073) (Azure Updates)
